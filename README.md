## How to Run

# Install http-server globally (if not already installed)
npm install -g http-server

# Run the server
http-server -p 8000
right-click on `index.html` or go to http://localhost:8000

- **HTML5** - Structure
- **CSS3** - Styling with gradients and responsive design
- **Vanilla JavaScript** - No frameworks, pure JavaScript
- **JSON** - Data storage

## Usage

1. **Search**: Type in the search box to find services by name or description
2. **Filter**: Click on category buttons to filter services by type
3. **Rate**: Click on the star rating to rate a service (you can only rate each service once)
4. **View Details**: Each service card displays name, phone number, description, category, and average rating
